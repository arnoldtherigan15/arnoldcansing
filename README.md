### How to Play
- Pilih lagu yang ada pada playlist
- Sentuh apapun untuk mulai bermain
- Click piano berdasarkan huruf yang ada di atas piano
- Salah huruf maka kamu harus bermain lagi dari awal



### Build using:
- HTML5
- CSS3
- JavaScript
